An AI-powered image classification system that detects **eczema (atopic dermatitis)** from skin images using deep learning techniques. This project aims to assist in early identification of eczema by analyzing skin images and providing predictions based on trained models.

---

## Overview

Eczema is a common inflammatory skin condition characterized by redness, itching, dryness, and irritation. Early detection can help prevent complications and improve treatment outcomes.

This project uses **computer vision and deep learning** to classify skin images and identify eczema patterns.

---

## Objectives

- Detect eczema from skin images using AI  
- Assist early identification and awareness  
- Demonstrate the application of deep learning in healthcare  
- Provide a simple and user-friendly prediction workflow  

---

## How It Works

1. User uploads a skin image.  
2. Image is preprocessed (resized, normalized, cleaned).  
3. Deep learning model analyzes visual patterns.  
4. Model predicts whether eczema is present.  
5. Prediction result is displayed.
